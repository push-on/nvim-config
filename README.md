## Neotree shortcuts
1. Backspace = go back directory
2. . = set current root
3. a = add file / folder
4. A = add folder
5. d = delete
6. r = rename
7. y = copy
8. p = paste
9. c = copy
10. m = move 
11. R = refresh
12. ? = help
13. < = previous source
14. > = next source
